REGEX PROJECT

This project uses regular expressions (regex) and python to find, validate, and extract data such as emails, usernames, phone numbers, and card numbers from a API data of customers. 

========================================
      FEATURES
========================================
1. sample_input.txt
   This file some customer data from our previous interactions. This is the data used to test our project on the effectiveness.

2. main.py
   This where I added the regex expressions and python to extract data from the file.

3. sample_output.json
   This file displays the extracted data saving it automatically after running the code.
